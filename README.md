# Health Clinic Appointment System

A comprehensive web application for managing clinic appointments. Built leveraging Spring Boot, JPA, H2 Database, Thymeleaf, and Spring Security.

## Project Description
This project streamlines the scheduling process for patients and health professionals. It provides dedicated dashboards for different types of users, complete with authentication and role based functionality. Patients can easily book appointments while doctors can manage their schedule. Administrators have full control over the platform users and support requests.

## Key Features
* User Authentication and Authorization
* Role Based Access Control (Admin, Doctor, Patient)
* Doctor Application and Approval Workflow
* Appointment Booking and Scheduling
* Dedicated Dashboards per Role
* Support Request Management
* Profile Management
* Responsive UI with Thymeleaf

## How to Run
1. Ensure you have Java 17 installed.
2. Open the project in your preferred IDE (like IntelliJ IDEA, Eclipse, or VS Code).
3. cd C:\Users\USER\Desktop\SE\Health-Clinic-Appointment-System\SE_Project\ClinicApp
4. ./mvnw.cmd spring-boot:run 
5. Open your web browser and navigate to http://localhost:8080
6. Login or register a new user account to explore the application.
